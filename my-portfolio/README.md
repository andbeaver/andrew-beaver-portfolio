# my-portfolio

Next.js 16 portfolio app for Andrew Beaver. See the [root README](../README.md) for full project details.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Environment Variables

Create `.env.local` in this directory:

```env
RESEND_API_KEY=your_resend_api_key
CONTACT_EMAIL=your@email.com
```

## Scripts

| Command | Description |
|---|---|
| `npm run dev` | Start development server |
| `npm run build` | Production build |
| `npm start` | Start production server |
| `npm run lint` | Run ESLint |

## Deploy

Deploy via [Vercel](https://vercel.com/new). Set the root directory to `my-portfolio` and add the environment variables in the Vercel dashboard.
